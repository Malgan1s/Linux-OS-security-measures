**2 Задания и вопросы**

**2.1 Опишем процесс установки дистрибутива Asra Linux.**

2.1.1 Процесс установки:

Установка Astra Linux выполняется через графический интерфейс или
терминал. Выбираем язык, раскладку клавиатуры, имя машины, имя
пользователя и пароль. Затем настраиваем жесткий диск (вручную или
автоматически) и устанавливаем дополнительные компоненты, например,
OpenSSH (опционально).

2.1.2 Настройка сетевых интерфейсов:

- Статические записи: Редактирование файла /etc/network/interfaces

- Получение IP по DHCP: В файле /etc/network/interfaces указать iface
  eth0 inet dhcp.

2.1.3 Обновление ОС: используем команды sudo apt update и sudo apt
upgrade.

2.1.4 Установка пакетов: используем команды sudo apt install
\<имя_пакета\>
